# Part 1 – Student Grade Tracker

## Description
This program processes raw student data, cleans it, analyzes marks, and generates summaries.

## Tasks Covered
- Task 1: Data cleaning and validation
- Task 2: Marks analysis and grading
- Task 3: Class summary table
- Task 4: String manipulation utility

## How to Run
1. Open terminal
2. Navigate to this folder
3. Run:
   python part1_grade_tracker.py

## Concepts Used
- Lists, dictionaries
- Loops (for, while)
- Conditional statements
- String operations
